# 📝 TodoList (Django)

🚀 Simple va qulay **TodoList web ilova**.
Foydalanuvchilar ro‘yxatdan o‘tib, login qilib, o‘z vazifalarini boshqarishlari mumkin.

---

## ✨ Features

* 🔐 Register / Login / Logout
* 📝 Task qo‘shish
* ✏️ Task tahrirlash
* ❌ Task o‘chirish
* ⏰ Notification time (eslatma vaqti)
* 📋 Barcha tasklarni ko‘rish

---

## 🛠️ Tech Stack

* **Backend:** Django (Python)
* **Database:** SQLite
* **Frontend:** HTML, CSS

## ⚙️ Installation

### 1️⃣ Clone project

```bash
git clone https://github.com/username/todolist.git
cd todolist
```

### 2️⃣ Virtual environment (ixtiyoriy)

```bash
python -m venv venv
```

**Linux / Mac:**

```bash
source venv/bin/activate
```

**Windows:**

```bash
venv\Scripts\activate
```

### 3️⃣ Dependencies o‘rnatish

```bash
pip install -r requirements.txt
```

### 4️⃣ Migratsiya

```bash
python manage.py migrate
```

### 5️⃣ Serverni ishga tushirish

```bash
python manage.py runserver
```
